🤖 Floating Chatbot Example

How to Use the Floating Chatbot

This example shows how to integrate the floating chatbot widget into your web application without npm installation.

📋 Integration Instructions

Step 1: Copy the source files to your project:

your-project/ ├── src/ │ ├── SmartChatbot.jsx │ ├── SmartChatbot.css │ └── llm.js └── index.html

Step 2: Import and use the component:

import SmartChatbot from './src/SmartChatbot.jsx'; import './src/SmartChatbot.css'; // Floating chatbot at bottom-right <SmartChatbot provider="openai" apiKey="your-api-key" model="gpt-4o-mini" isFloating={true} position="bottom-right" theme="light" />

Available Positions:

  • bottom-right - Bottom right corner (default)
  • bottom-left - Bottom left corner
  • top-right - Top right corner
  • top-left - Top left corner

Features:

  • 💬 Floating chat bubble that expands when clicked
  • 🎨 Light and dark themes
  • 📱 Responsive design for mobile devices
  • 💾 Conversation persistence with localStorage
  • 🔧 Support for OpenAI, Anthropic, and Gemini
💡 Tip: The floating chatbot will appear as a chat bubble icon. Click it to expand the full chat interface. Use the minimize button (➖) to collapse it back to the bubble.
